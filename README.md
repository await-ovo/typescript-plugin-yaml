<p align='center'>A Typescript Language Service plugin for YAML.</p>



## Usage

```shell
# With PNPM
pnpm add typescript-plugin-yaml -D

# With Yarn
yarn add -D typescript-plugin-yaml
```

And then add this to `tsconfig.json`.
```json
{
  "compilerOptions": {
    "plugins": [{ "name": "typescript-plugin-yaml" }]
  }
}
```
